Tetris

How to Play Tetris

  • Tetris is a game about stacking falling blocks.
  • The goal is to fill complete horizontal rows which will then disappear and give you points.
  • Use arrow keys ← → to move blocks left and right.
  • Use the up arrow key ↑ to rotate blocks.
  • Use the down arrow key ↓ to speed up block descent.
  • Use the space bar to drop blocks instantly to the bottom.
  • The game ends when blocks reach the top of the screen.
